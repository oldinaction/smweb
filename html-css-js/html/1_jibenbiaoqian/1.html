一、html语法简介

1、html标签和元素:html标签是组成html元素的，如一对body标签就是一个元素，一对p标签组成的也是一个元素

2、html标签是由尖括号包围的关键词，比如 <html>

3、html标签通常是成对出现的,也有单个的

4、html的结构包括头部（head）、主体（body）两大部分

5、>!DOCTYPE>标签，如：<!doctype html>

6、html属性和属性值，如：<body bgcolor="red"> bgcolor就是body标签的一个属性

7、书写注意事项:字母最好使用小写，大写也可以实现；属性值通常使用双引号包围，单引号也可以实现

1、段落标签：<p> </p> 他会自动换行

2、标题标签：

注意事项：h1在一个html文档里面只能使用一次，这样有利于SEO优化

<h1> </h1>

<h2> </h2>

<h3> </h4>

<h4> </h4>

<h5> </h5>
<h6> </h6>

3、换行标签：< br />

4、注释标签：<!-- -->

5、预格式标签：<pre></pre>

<pre>
<p>
<b>预格式1</b>
<strong>预格式2</strong>
</p>
</pre>

效果如下：（前面的缩进也会保留）
		<p>
			<b>预格式1</b>
			<strong>预格式2</strong>

三、html实体

小于号： &lt;     效果：<

大于号： &gt;     效果：>

引号： &quot;     效果："

空格： &nbsp;

版权号： &copy;     效果：©

四、html排版标签

1、font标签：<font></font> 规定文本的字体、字体尺寸、字体颜色。标签属性如下:

size : 12,13 ...

color : red,blue,green,yellow 或者 颜色值#333

face : "黑体" "微软雅黑" "宋体"

2、strong/b 加粗标签

<b> </b>   和   <strong> </strong> 都有对文本加粗效果

但是strong用于强调文本，它强调的程度更强一些，表示重要文本，对SEO优化有用。

五、html超链接标签

<a> </a>

A、一般链接：链接到我的博客

B、锚链接，链接到本页面:

==>首先创建一个锚链接的锚记点:     <a href="#锚记点名称">主题名称</a>

==>然后在具体要链接到的地方设置name:     <a name="锚记点名称"></a>    （锚记点前面要加上#号）

==>有时候页面高度不够，点击锚链接后，要链接到的位置不会显示在浏览器的第一行，因为此时页面的底部已经全部显示出来了。此时可以增加body高度实现。

C、电子邮件链接:

<a href="mailto:邮件地址">站长邮箱</a>(注意不能忘了mailto) 站长邮箱

六、marquee滚动标签

<marquee></marquee>    默认占一行

属性:文字移动方向：  direction：left、right

<marquee direction="left">向左飘</marquee>

<marquee direction="right">向右飘</marquee>

效果如下：
向左飘 向右飘

七、frameset框架标签

示例代码：
		<frameset rows="10%,*">
			<frame src="top.html" noresize="noresize" />
			<frameset cols="16%,*">
				<frame src="left.html" />
				<frame src="content.html" name="show" />
			</frameset>
		</frameset>
其中left主要代码：
	<a href="html/0/0.html" target="show">0 - Html - Html模板</a>
	<a href="html/1/1.html" target="show">1 - Html - 基本标签</a>
注意：

1、frameset中含有frame

2、此时frameset代替了body，有frameset就不写body，frameset一般用于后台管理，在前台页面不利于SEO优化，src对应的目录下要有相对应的文件

3、上面的rows="10%,*"是将框架分为两行，左边为总宽度的10%，右边为剩下的

4、name="show"和target="show"的利用，表示点击左边框架的left页面中的链接，链接到的页面会显示在右边框架里，如果没有则在左边框架中显示

5、noresize="noresize"表示不能改变框架大小
