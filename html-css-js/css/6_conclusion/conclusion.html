1、消除a连接的下划线和img边框时，必须写在对应的a和img标签上

（1）若是文字连接：a{text-decoration:none;}（要写在对应的a标签上）

（2）若是图片连接：img{border:0;}（要写在对应的img标签上）

示例：
			<div style="float:left;text-decoration:none;border:0;">
			<a href="#">文字链接</a>
			<a href="#" style="border:0;"><img src="images/1.jpg"/></a>
			</div>

			<div style="float:left;">
				<a href="#" style="text-decoration:none;">文字链接</a>
				<a href="#"><img src="images/1.jpg" style="border:0;"/></a>
			</div>
			<div style="clear:both;"></div>

2、hover相关的写法

如：a:hover{}    a:hover img{}    span:hover{}

示例：
			<style type="text/css">
				.s2 .s2_1:hover{text-decoration:none;}
				.s2 .s2_2:hover img{border:2px solid red;}
				.s2 span:hover{font-weight:900;}
			</style>
			------------------------------------
			<div class="s2">
				<a href="#" class="s2_1">鼠标滑过去掉下划线</a>
				<a href="#" class="s2_2"><img src="images/1.jpg"/></a>
				<span>鼠标滑过字体变粗</span>
			</div>
鼠标滑过去掉下划线 鼠标滑过字体变粗

3、居中对齐(margin:0 auto;)使用时要给定宽度

居中对齐(margin:0 auto;)，必须给定宽度，宽度可为%形式，高度无要求（如果不写，子元素必须可以把这个容器撑起来）。

示例：
			<style type="text/css">
				.s3 .s3_1{width:400px;margin:0 auto;background:#F00;}
				.s3 .s3_2{width:400px;margin:0 auto;background:#F9C;}
				.s3 .s3_3{width:200px;height:20px;margin:0 auto;background:#3F3;}
				.s3 .s3_4{width:40%;height:20px;margin:0 auto;background:#09C;}
			</style>
			------------------------------------
			<div class="s3">
				<div class="s3_1"></div>
				<div class="s3_2">上面还有个div，但是里面没有其他元素把他撑起来，也没指定高度，所以没有显示出来</div>
				<div class="s3_3"></div>
				<div class="s3_4"></div>
			</div>
上面还有个div，但是里面没有其他元素把他撑起来，也没指定高度，所以没有显示出来
