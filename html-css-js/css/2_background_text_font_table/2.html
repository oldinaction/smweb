CSS笔记 - background、text、font、table - 小易

一、background背景属性

1、背景颜色 background-color:3种颜色值;

2、背景图片 background-image:url(); url中的图片路径可以加双引号也可以不加，若是写在内联样式中则要写成单引号或者不写

3、重复 background-repeat : repeat/no-repeat/repeat-x/repeat-y 默认是repeat

4、位置 background-position:垂直方向位置 水平方向位置（两个位置颠倒也可以实现效果）

（1）垂直方向：top center bottom 水平方向：left center right

（2）百分比形式

（3）像素形式

位置是相对容器

background-position必须要在background-image存在的条件下才能使用

网页上的小图标一般是通过改变一张大图的位置来显示大图上的小图标

5、大小 background-size : auto/cover/contain

6、简写与标准写法：建议使用简写

（1）简写 background:url(bg.png) no-repeat top center;每个属性值之间一定要有空格

（2）标准写法 background-image:url(bg.png);background-repeat:no-repeat;background-position:top center;

二、文本

1、文本颜色 color （于backgound-color不同，他只针对文本的颜色）

2、字符间距 letter-spacing : 10px/1em;（10px=1em）

3、行高 line-height : 20px/150%/2（2代表200%，默认为100%）

4、文本对齐(水平方向) text-align:left/center/right

（注）垂直方向对齐 : 让line-height的值为容器的的高度（仅适用于文本垂直居中）

5、缩进 text-indent：20px（默认为0，即顶格）

6、装饰文本 text-decoration 可简写

（注）其中常用text-decoration-line : none/underline/overline(上划线)/line-through(贯穿线)/blink(闪烁)

7、文本转换 text-transform : capitalize(所有单词首字母大学)/uppercase(大写)/lowercase(小写)

8、text-overflow：clip(裁剪)/ellipsis(省略，显示省略号)

注：要想将多余的文本显示未省略号，width、overflow:hidden、white-space:nowrap、text-overflow:ellipsis几个属性都要有

三、字体

1、font-size:12px;（统计表明12px的字体是人们看着最舒服的字体大小）

2、font-family : "宋体"/"微软雅黑";

3、font-style : none(默认)/italic(斜体)/oblique(倾斜);

4、font-weight : normal(400)/bold(粗体 700)/bolder(特粗体 相当b和strong)/lighter(细体)/100~900;

四、表格

1、边框 border : 1px solid red; 调试常用

2、去掉折叠边框 border-collapse:collapse;

3、文字居中 text-align:center; line-height:30px;

4、鼠标停留选中变色 .table tr td:hover{background:#666;}
